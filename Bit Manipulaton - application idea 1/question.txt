Description

for a given array arr with n elements calculate arr[i]^arr[j] where i,j>=0 & i,j<n
for all possible pairs.

^ means XOR


Input Format

The first line of input contains t - the number of test cases.
Next, t lines contain array size n followed by n elements.


Output Format

output single integer for each case

Constraints

n >=10^5

Sample Input 1

2 
3 
1 5 7
4 
2 3 5 11

Sample Output 1

12
45
